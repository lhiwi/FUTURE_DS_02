# FUTURE_DS_02 — Customer Retention & Churn Analysis

## Objective
Analyze churn patterns, retention drivers, and customer lifetime trends using the Telco Customer Churn dataset, and present actionable recommendations for a subscription-style business.

## Tools
- Google Colab (Python)
- Power BI (Dashboard)
- GitHub (Documentation)

## Repo Structure
- data/ (dataset link + optional sample)
- notebooks/ (Colab notebooks)
- dashboard/ (Power BI .pbix + exports)
- reports/ (final report + screenshots)

## Key Questions
1. Why are customers leaving?
2. Which segments churn the most?
3. How long do customers stay?
4. What actions improve retention?

## How to Reproduce
1) Download dataset into `data/` (see `data/DATA_SOURCE.md`)
2) Run notebooks in `notebooks/`
3) Open Power BI file in `dashboard/`
